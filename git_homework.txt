1. Заходим в терминале каталог проекта ..../Git Homework/
2. вводим команду git init
3. создаем файл .gitignore в корне проекта
    3.1. Вносим данные в файл:
        *.sass
        /bin
        /admin
        /config
4. git add .

!!! расхождение с видеоуроком!!!
!!! git не создает новый каталог при создании ветки, а только указатель!!!
!!! так же git не создает пустые папки!!!
!!! поэтому создание файлов идет через копирование!!!

5. создаем папку blog с файлами index.html и index.js в ..../Git Homework/
6. Делаем комит изменений

7. создаем новую ветку: git branch homework
8. копирую каталог с файлами в новую ветку git checkout master blog/*.*
